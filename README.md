**Lecteurs** est une application de bureau (Windows) pour **collectionner, organiser et lire** vos images, illustrations, mangas et webtoons. Tout reste sur votre machine : **aucun compte, aucune donnée envoyée en ligne**, et l'IA tourne **localement**.

L'application s'installe **vide** : c'est un organiseur et un lecteur pour **vos propres fichiers**.

## ⬇️ Télécharger

➡️ **[Dernière version (installateur Windows)](https://github.com/Krnight/lecteurs-releases/releases/latest)**

Windows 10/11 (64-bit). Installation classique, puis **mises à jour automatiques** (avec un écran « Quoi de neuf » à chaque nouveauté).

## ✨ Fonctionnalités

### 📚 Bibliothèque & organisation
- Ajout de dossiers par **glisser-déposer**, avec **sous-dossiers récursifs**.
- **Tags manuels**, **favoris**, recherche par nom ou par tag.
- Dossiers **personnalisables** : couleur, icône et image de fond (y compris **GIF / WebP animés et vidéos**), épinglage, masquage du contenu sensible (+18).
- **Catégories** visuelles de tags (regroupement automatique : cheveux, yeux, vêtements, séries…).
- **Détection de doublons** par empreinte perceptuelle (sans IA), avec suppression des doublons en un clic.
- **Actualiser** : détecte les fichiers ajoutés, renommés ou supprimés sur le disque.
- **Import d'images** directement dans un dossier par glisser-déposer.

### 📖 Lecteur de séries (manga / webtoon)
- Crée des **Séries** reliées à un dossier ; les **chapitres** sont déduits automatiquement des sous-dossiers.
- **Lecteur immersif** plein écran, avec interface adaptée au mode :
  - **Vertical** (défilement webtoon, largeur réglable) ou **Horizontal** (page par page),
  - **sens** gauche → droite ou droite → gauche (manga) — **mémorisé par série**.
- **Enchaînement automatique** des chapitres, **sélecteur de chapitres à vignettes**, réglette de navigation.
- **Reprise automatique** exactement là où vous vous êtes arrêté.
- **Fiche série** soignée avec **fond personnalisable** (la couverture floutée par défaut).
- **Espace Séries** dédié : section **« Reprendre »**, filtres (en cours / terminées / non commencées / favoris), **recherche** et **genres** (que vous définissez vous-même).

### 🤖 IA locale (optionnelle, téléchargée à la demande)
- **Auto-tagging** d'illustrations façon *booru* (personnages, cheveux, vêtements, poses…) via un modèle dédié à l'anime.
- **Recherche en langage naturel** (« fille aux cheveux blancs dans la neige ») grâce à un modèle local.
- **OCR** : reconnaissance du texte dans les images (utile pour le scantrad).
- **Upscale** (Real-ESRGAN anime) — agrandit vos images, en **préservant l'animation** des WebP/GIF animés.
- **Traduction des tags** en français (ou autre), en local.
- **Workflow scantrad** : suivi par page (brut → nettoyé → édité → relu → terminé).

> Tous les modèles se téléchargent **une seule fois** puis fonctionnent **hors-ligne**. Rien n'est envoyé sur Internet.

### ♿ Accessibilité
- Contraste élevé, **police adaptée à la dyslexie** (OpenDyslexic), agrandissement de l'interface, **mode daltonien**.

### 🔒 Confidentialité
- **100 % hors-ligne** : aucune télémétrie, aucun compte, aucune donnée envoyée.
- Vos fichiers restent vos fichiers — l'app ne fait que les organiser et les afficher.

## 🔄 Mises à jour

L'application se met à jour **toute seule** : elle vérifie ce dépôt, télécharge la nouvelle version et l'installe, puis vous présente les nouveautés.

## 🖥️ Configuration recommandée

- **Windows 10/11** 64-bit.
- Un **GPU récent** accélère les fonctions IA (upscale, tagging), mais le CPU fonctionne aussi.

## ℹ️ À propos de ce dépôt

Ce dépôt public héberge uniquement les **téléchargements et les fichiers de mise à jour automatique**. Le code source est privé.
